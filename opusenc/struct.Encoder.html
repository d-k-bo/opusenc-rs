Struct opusenc::Encoder

pub struct Encoder { /* private fields */ }
Expand description

Encoder for creating Ogg Opus files.

Create an encoder using Encoder::create_file, write samples using Encoder::write/Encoder::write_float and make sure to call Encoder::drain before dropping it.
